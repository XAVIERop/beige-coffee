## Tips before sending

1. **Personalize** — Use their first name. If you’ve visited, mention a drink or dish you liked.
2. **Link** — Deploy the site (Vercel, Netlify, GitHub Pages) and use the live URL.
3. **Follow-up** — If no reply in 5–7 days, send a short follow-up: “Hi [Name], just bumping this in case it got buried. The demo link is still live if you’d like to take a look.”
4. **Keep it light** — Avoid jargon. Focus on what they get, not tech details.
